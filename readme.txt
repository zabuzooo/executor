用途↓
JSONファイルの変更を検知して、PHPMailerを使用しメールを送信するバッチプログラムです。
開発途中の為、動作不十分な点もありますが、ポートフォーリオとして一旦アップしておきます。

使用方法↓
①config.yaml　に設定を追加、コメントアウトされている内容を入力します
②execute_files.yaml　に監視対象のファイル名またはディレクトリ名を入力します、対象が増えた場合はyaml形式で下に追加していってください。
③execute.php　をバッチとしてcron等で定期実行

Mailer.phpのメール設定部分にメールが送信されます
想定していた動作は、変更が検知されたら当プログラムを組み込んだプログラムが停止するように考えていました。
config.yaml にメールアドレスを定義したほうが圧倒的に楽ですのでまたの機会に修正します。
